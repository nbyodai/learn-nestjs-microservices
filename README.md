# Learning Microservices with NestJS

A tutorial following showcasing microservices in nestjs

## Description
This repository contains example services and configuration for learning NestJS microservices, message patterns, and local development workflows.

### Topics Handled
[HTTP, NATS, RabbitMQ, Request-Response, Event-Patterns, Message-Patterns, Docker]

## Quick Start

### Prerequisites
- Node.js (LTS)
- npm or yarn

### Install
```bash
npm install
# or
yarn
```

### Run (development)
```bash
npm run start:dev
# or
yarn start:dev
```
